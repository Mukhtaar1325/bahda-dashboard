# BAHDA Dashboard Project

Data pipeline and interactive dashboard for visualizing KoboToolbox survey field data.

## Initial Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate and install dependencies: `pip install -r requirements.txt`
3. Edit `config/secrets.env` with your Kobo API keys.
4. Run the dashboard: `streamlit run app/streamlit_app.py`
